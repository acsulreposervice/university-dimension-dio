# university-dimension-dio

## 📋 Sobre o Projeto

Atividade proposta no módulo sobre modelagem de dados 
usando a abordagem dimensional.

A atividade girou em torno da transformação
de um modelo relacional para o modelo dimensional,
tendo como tabela fato, informações relacionadas a
professores em uma universidade.


### 🔍 Desafios Encontrados

Os principais desafios se mostraram na implementação
da tabela de datas, onde foi necessário revisar para
a realização da melhor forma de implementar o relacionamento
entre as tabelas. Porém, no geral, serviu para o aprendizado
do contexto temporal dentro do domínio da atividade.

## 🛠️ Tecnologias Utilizadas

Foram utilizadas as seguintes tecnologias:

- **MySQL**: para a criação, população e implementação dos modelos de dados.
> *university_dio_ddl.sql*
> *university_dio_dml.sql (esta sendo criada via IA com dados ficticios).*

- **PowerBI**: para a realização de transformações pontuais, criações de tabelas
extras e relacionamentos.
> *university_dimension_dio.pbix*
